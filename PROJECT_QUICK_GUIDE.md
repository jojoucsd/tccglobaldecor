## 2. File Names

**Use numbers only:** `0.jpg`, `1.jpg`, `2.jpg`, `3.jpg`...

- Start at 0 (not 1)
- No skipped numbers
- Formats: `.jpg`, `.jpeg`, or `.png`

---

## 3. What Each Photo Should Be

| # | Photo Type |
|---|------------|
| 0 | Building exterior (cover) |
| 1 | Wide banner (top of page) |
| 2 | Interior with carpet |
| 3 | Floor plan/drawing |
| 4 | Another interior |
| 5+ | Extra photos (optional) |

**Minimum:** 5 photos (0-4)
**Recommended:** 7+ photos
