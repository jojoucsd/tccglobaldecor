# 如何提交新專案

**給團隊的簡易指南 — 無需技術背景！**

---

## 步驟 1：建立專案資料夾

**資料夾名稱必須：**
- 只用小寫字母
- 用短橫線（`-`）取代空格
- 不使用特殊符號（不要用 .、&、' 等）

**範例：**
- "Octavius Tower Caesars Palace" → `octavius-caesars`
- "Park Hyatt Niseko Hanazono" → `park-hyatt-niseko`
- "Grand Hyatt Macau" → `grand-hyatt-macau`
- "The Londoner Macau" → `londoner-macau`

---

## 步驟 2：準備照片

### ⚠️ 重要 — 檔案格式與命名：

**你可以：**
- ✅ 依需要調整大小、裁切或編輯照片
- ✅ 使用 `.jpg`、`.jpeg` 或 `.png` 格式

**你必須：**
- ✅ 只用數字正確重新命名檔案（見下方說明）
- ✅ 使用正確的檔案格式（`.jpg`、`.jpeg`、`.png`）

---

### 照片要求

**最少：5 張照片**（編號 0–4）
**建議：7 張照片**（編號 0–6）
**最多：不限**（繼續編號：7、8、9、10、11、...）

---

### 每張照片應該是什麼

| 檔案名稱 | 照片內容 | 說明 |
|---------|---------|------|
| **0** | 建築外觀照片 | **建議直式** — 這是封面圖片 |
| **1** | 寬幅橫向橫幅 | 顯示在專案頁面頂部 |
| **2** | 實際室內照片 | 已安裝地毯的實景 |
| **3** | 圖面或平面圖 | **與照片 2 同一區域**的平面圖/圖面 |
| **4** | 另一個重要室內空間 | 主要空間、大廳、走廊等 |
| **5** | 細節或特寫 | 地毯材質、圖案、邊緣細節 |
| **6** | 最後一張主要室內或細節 | 最後一張重點照片 |
| **7+** | （選用）額外照片 | 其他角度、細節或區域 |

---

### 檔案命名規則

**只用數字 — 不要用字母、不要用文字**

✅ **正確：**
```
0.jpg
1.png
2.jpg
3.jpg
4.jpg
5.png
6.jpg
```

❌ **錯誤：**
```
exterior.jpg
banner photo.png
photo_2.jpg
IMG_1234.jpg
```

**規則：**
- 從 0 開始（不是 1）
- 數字不可跳號（0、1、2、3 ✅ / 0、1、3、5 ❌）
- 檔名不可有空格
- 允許的格式：`.jpg`、`.jpeg`、`.png`

## 步驟 3：實際範例

### 範例 1：Grand Hyatt Macau（5 張照片）
**資料夾：** [grand-hyatt-macau](https://github.com/jojoucsd/tccglobaldecor/tree/main/public/images/projects/grand-hyatt-macau)

```
├── 0.avif     ← 建築外觀（封面圖片）

```

**說明：** 此專案有 5 張照片，為最少數量。網站將顯示 A 和 C 區段。

## 步驟 4：提供專案詳細資訊（選用）

**說明：** 此步驟為選用。有時照片提供者不需要提供專案說明和細節 — 這可以由負責實作的人員處理。但如果您能提供這些資訊會非常有幫助！

如果您有這些資訊，請複製並填寫此範本：

```
PROJECT NAME（專案名稱）: Octavius Tower Caesars Palace
FOLDER NAME（資料夾名稱）: octavius-caesars

ADDRESS（地址）:
3570 S Las Vegas Blvd, Las Vegas, NV 89109, USA

SUMMARY（摘要）（2–3 句）:
簡要概述該物業 — 其設計理念、氛圍或獨特特色。

DESCRIPTION（描述）（3–4 句）:
TCC Carpets 如何為專案做出貢獻。我們創造了什麼？使用了哪些技術或材料？
地毯如何提升空間？

NOTES（備註）（選用，1–2 句）:
關於圖案、顏色或地毯安裝特殊功能的額外細節。
```

**實際範例：**

```
PROJECT NAME: Park Hyatt Niseko Hanazono
FOLDER NAME: park-hyatt-niseko

ADDRESS:
328-47 Aza Iwaobetsu, Kutchan, Abuta District, Hokkaido, Japan

SUMMARY:
坐落於 Annupuri 山腳下，Park Hyatt Niseko Hanazono 透過寧靜奢華與精緻工藝
展現自然之美。室內設計靈感來自周圍景觀——雪、木頭和石頭。

DESCRIPTION:
在此背景下，TCC Carpets 設計了一系列訂製地面材料，反映大自然的韻律和
日本克制的美學，在視覺寧靜與觸覺溫暖之間取得平衡。

NOTES:
微妙的漸層和有機圖案與套房和公共區域的木材和石材內裝相得益彰。
```

## 步驟 5：提交您的專案

**專案準備好後：**

1. 將所有照片放入名稱正確的資料夾（小寫字母加短橫線）
2. 確認檔案命名為：`0`、`1`、`2`、`3`、`4` 等（不可跳號）
3. 透過以下方式傳送給網站開發人員：
   - 上傳至雲端儲存空間（Google Drive、Dropbox 等），或
   - 以 zip 檔案透過電子郵件傳送

**就這樣！** 開發人員會處理其餘工作。

## ❓ 常見問題

**Q：如果我沒有照片 3 的圖面怎麼辦？**
A：改用其他室內照片。但理想情況下，如果有平面圖或技術圖面請提供。

**Q：我可以用手機照片嗎？**
A：可以！只要清晰且光線充足即可。

**Q：如果我有超過 7 張照片怎麼辦？**
A：太好了！繼續編號：7、8、9、10、11 等。

**Q：如果我只有 5 張照片怎麼辦？**
A：沒關係 — 只要傳送您有的照片（0、1、2、3、4）。

**Q：照片需要特定大小或格式嗎？**
A：沒有特定要求 — `.jpg`、`.jpeg` 或 `.png` 格式都可以。開發人員會處理最終優化和轉換。
