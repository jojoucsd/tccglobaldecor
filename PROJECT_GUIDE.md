## STEP 1: Create a Project Folder

**Folder name must be:**
- Lowercase letters only
- Use dashes (`-`) instead of spaces
- No special characters (no ., &, ', etc.)

- "Octavius Tower Caesars Palace" ‚Üí `octavius-caesars`

### Photo Requirements

**Ideally provide 7 photos** (numbered 0‚Äì6)

---

### What Each Photo Should Be

| File Name | What It Shows | Notes |
|-----------|---------------|-------|
| **0** | Exterior building photo | **Portrait preferred** ‚Äî this is the cover image |
| **1** | Wide landscape banner | Shows at the top of the project page |
| **2** | Real interior photo | Actual installed carpet in the space |
| **3** | Drawing or plan view | Floor plan/drawing of **the same area as photo 2** |
| **4** | Another important interior | Main space, lobby, hallway, etc. |
| **5** | Detail or close-up | Carpet texture, pattern, edge detail |
| **6** | Final main interior or detail | Last key photo |
| **7+** | (Optional) Extra photos | Any additional images you want to include |

---

### File Naming Rules

**Numbers only ‚Äî no letters, no words**

‚úÖ **Correct:**
```
0.jpg
1.png
2.jpg
3.jpg
4.jpg
5.png
6.jpg
```

‚ùå **Wrong:**
```

```

**Rules:**
- Start at 0 (not 1)
- No skipped numbers (0, 1, 2, 3 ‚úÖ / 0, 1, 3, 5 ‚ùå)
- No spaces in names
- Allowed formats: `.jpg`, `.jpeg`, `.png`

## ‚ùì Questions?

**Q: What if I don't have a drawing for photo 3?**

**Q: Can I use my phone photos?**
A: Yes! As long as they're clear and well-lit.

**Q: What if I have more than 7 photos?**
A: Great! Keep numbering: 7, 8, 9, 10, 11, etc.

**Q: What if I only have 5 photos?**
A: That's okay ‚Äî just send what you have (0, 1, 2, 3, 4).
